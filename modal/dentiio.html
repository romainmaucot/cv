Dentiio

Projet entreprenarial



Une plateforme open source de partage de connaissances bucco-dentaires à destination des chirurgiens-dentistes et étudiants.

Le front end est développé avec React.
Le back end est une API en Symfony 4. Elle est documentée avec Swagger.
L'intégration continue est gérée avec GitActions, le déploiement continu avec Heroku.
Docker à été mis en place afin de gérer l'environnement de l'application.
